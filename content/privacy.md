# Privacy Policy

This is a placeholder Privacy Policy page.

## For Template Users

Replace this URL in `AppConfiguration.swift` with your actual Privacy Policy before submitting to the App Store.

```swift
static let privacyPolicyURL = "https://yourapp.com/privacy"
```

## Requirements

Apple requires all apps to have a Privacy Policy that describes:

- What data you collect
- How you use the data
- Third-party services (analytics, crash reporting, etc.)
- User rights and data deletion

## Template Default Services

If using the default configuration, your privacy policy should mention:

- **RevenueCat** - Subscription management
- **TelemetryDeck** - Privacy-preserving analytics (no PII collected)
- **Firebase** (if enabled) - Authentication and data storage
